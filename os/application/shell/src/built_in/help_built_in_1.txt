Built‑In Commands (1 of 2):

  alias [KEY=VALUE]
      List all aliases, or define KEY to VALUE.
      Example: alias hhu='Heinrich Heine Universitaet'

  unalias KEY
      Remove the alias named KEY.
      Example: unalias hhu

  cd [DIR]
      Change directory to DIR.
      `./` = current, `../` = parent.
      Example: cd ../projects

  clear
      Clear the terminal screen.
      Example: clear

  echo [ARGS…]
      Print ARGS to stdout followed by newline.
      Example: echo "Hello, world!"

Type `help controls` to see navigation keys.
Type `help tokens`   to see special symbols.
Type `help built‑in‑1` or `help built‑in‑2` for built‑ins.
