Usage: shell [OPTIONS]
An interactive shell featuring:
  • Line editing
  • Command history
  • Tab‑completion
  • Built‑in commands

Options:
  --no-auto-completion   Disable optional auto-completion module
  --no-history           Disable command history module

Type “help controls” to see navigation keys.
Type “help tokens”   to see special symbols.
Type “help built‑in‑1” or “help built‑in‑2” for built‑ins.
