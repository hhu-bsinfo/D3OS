Special tokens:
  '    Single‑quotes
  "    Double‑quotes
  |    Pipe: send output of left command into right
  <    Redirect input from file
  >    Redirect output (truncate)
  >>   Redirect output (append)
  &    Run command in background
  ||   Logical OR: run next if first fails
  &&   Logical AND: run next if first succeeds
  ;    Command separator

Examples of Pipelines & Redirections:
  $ grep "error" < input.txt | sort >> output.log
